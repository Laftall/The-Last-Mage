# The-Last-Mage

Jogo cooperativo local para 2 jogadores, onde os jogadores enfrentam inimigos utilizando diferentes magias e habilidades especiais.

Requisitos
2 controles conectados ao computador
Projeto desenvolvido na Godot Engine
CONTROLES

Controle PlayStation

Atirar: R2
Habilidade Especial: R1
Pegar Poção: Quadrado (□)
Usar Poção: L2
Movimentação: Analógico Esquerdo
Mirar: Analógico Direito

Controle Xbox

Atirar: RT
Habilidade Especial: RB
Pegar Poção: X
Usar Poção: LT
Movimentação: Analógico Esquerdo
Mirar: Analógico Direito

Observação:
O jogo requer 2 jogadores, cada um utilizando um controle compatível (PlayStation ou Xbox).
